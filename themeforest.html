Welcome to Kayzen; the next generation of website & app theming. Kayzen is more than just a theme, it's a theme framework, custom built from the ground up to help make theming great again.

Give your visitors an experience to remember - Kayzen is a modern theme framework with 12 premade demos which are built to be engaging and interactive. Create a stunning website simply and quickly thanks to our clean and intuitive code or simply use the provided page builder. We have built Kayzen using our own custom framework, making it eaiser for you to manage, and quicker for you to build.

Main Features

  • Built using our own modular Sass framework
  • Drag & drop page builder
  • Font Awesome icons
  • Unlimited colors
  • 6 unique homepage designs
  • 100% responsive built using mobile-first principles
  • Built using our own grid system
  • Built with Sass following modular CSS principles
  • Mega-menu and infinitely nestable dropdown menus
  • Mobile friendly flyout navigation
  • All pages 100% valid HTML5
  • 200+ HTML files
  • Includes multi-page/one-page options
  • Working contact form with HTML5 form validation
  • Working Twitter feed
  • Scroll animations
  • Customizable Google map
  • Parallax effects
  • Video background
  • Sticky-header option
  • Only free Google Fonts used
  • Source PHP template files included
  • Owl Carousel with touch-swipe support
  • Magnific Popup responsive jQuery lightbox
  • Complete and comprehensive documentation
  • Easy to customize and build upon
  • Hosted on Github - gain access to private repository and free support
